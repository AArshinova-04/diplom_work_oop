# Резервное копирование картинок с котиками

Программа для резервного копирования картинок с сайта cataas.com на Яндекс.Диск.

## Требования
- Python 3.7+
- Токен Яндекс.Диска (получить на [Полигоне Яндекс.Диска](https://yandex.ru/dev/disk/poligon/))

## Установка

1. Клонируйте репозиторий
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
